Collaborative Creative Coding with p5js & Ably

Each sketch publishes events (mouse coordinates or clicks) that other sketches respond to.

  • Sketch 1 - controls size & brightness for sketch 2 & 3
  • Sketch 2 - controls position for sketch 1
  • Sketch 3 - controls jitter for sketch 1

Using the sketches collaboratively requires an Ably API key with `publish` and `subscribe` capabilities. Create a free account at ably.com.
